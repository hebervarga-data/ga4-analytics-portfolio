# GA4 Analytics Portfolio Project  
# Proyecto de Portfolio – Analítica GA4

---

## Project Overview | Descripción del Proyecto

**EN**  
This project demonstrates an end-to-end analytics workflow using GA4 event data, transforming raw data into business-ready KPIs and dashboards.

**ES**  
Este proyecto muestra un flujo completo de analítica utilizando datos de eventos GA4, transformando datos crudos en KPIs listos para negocio y dashboards interactivos.

---

## Tech Stack | Tecnologías

- BigQuery (Data Warehouse)
- SQL (Data Modeling)
- Python (Automation)
- Power BI (Visualization)
- DAX (Business Metrics)
- GitHub (Version Control)

---

## Project Goal | Objetivo

**EN**  
Design a scalable analytics pipeline that separates raw data from business metrics, ensuring consistency and performance.

**ES**  
Diseñar un pipeline de analítica escalable que separe datos crudos de métricas de negocio, asegurando consistencia y buen rendimiento.
